# Natural Event Tracker (NET)

NET is a system output for System Architecture and Integration Course (TUP-T 2021).

Technologies used:
- Laravel 8
- API's (Earth Observatory Natural Event Tracker API (NASA), News API, & Global Giving API)
- Bootstrap
- Fontawesome
- MySQL
- jQuery
